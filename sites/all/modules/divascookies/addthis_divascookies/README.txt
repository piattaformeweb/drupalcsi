DESCRIPTION
-----------

Implements hook_divascookies_script_stop() for the Addthis module 
(https://www.drupal.org/project/addthis) for use with Divas Cookies.

REQUIREMENTS
------------

 * Addthis (>7.x-4.0 alpha)

INSTALLATION
------------

 1. ENABLE THE MODULE

    Enable the module on the Modules admin page.


CONFIGURATION
-------------

No configuration needs.
